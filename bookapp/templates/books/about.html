About Watbooks

A Repository for Tracking Books

This site was developed to keep track of my ever growing collection of books. Continuous book purchases have often left me wondering in-store whether I already owned a book.

This site allows me to quickly check whether or not I already own a book to avoid buying duplicates. I also plan to add features which will allow me to track which books I have lent to friends.

A Web-App Building Exercise

I am also using this site to practice and further develop my coding skills. I have used the Python-based Django web development framework to build the back-end of the site, as well as HTML and CSS to build the front-end. As I continue to learn I plan to add to this site, so keep an eye out for changes.
